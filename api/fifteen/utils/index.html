fifteen.utils.loop_metric_generator() → Generator[LoopMetrics, None, None][source]

Generator for computing loop metrics.

Note that the first iteration_per_sec metric will be 0.0.

Example usage:

# Note that this is an infinite loop.
for metric in loop_metric_generator():
    time.sleep(1.0)
    print(metric)

or:

loop_metrics = loop_metric_generator()
while True:
    time.sleep(1.0)
    print(next(loop_metrics).iterations_per_sec)

fifteen.utils.pdb_safety_net() → None[source]

Attaches a “safety net” for unexpected errors in a Python script.

When called, PDB will be automatically opened when either (a) the user hits Ctrl+C or (b) we encounter an uncaught exception. Helpful for bypassing minor errors, diagnosing problems, and rescuing unsaved models.

Approximately emulates launching a script via python -m pdb -c continue script_name.py.
